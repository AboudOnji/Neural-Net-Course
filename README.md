## Overview
Welcome to the Neural Net Course repository. This collection serves as a comprehensive resource for understanding the fundamentals and advanced concepts of Neural Networks and Computational Intelligence. The materials included here cover theoretical foundations, mathematical models, and practical implementations.

## Course Content
The course covers a wide range of topics, structured to guide you from basic concepts to advanced architectures:

- **The Artificial Neuron**: Understanding the basic building block of neural networks.
- **Supervised Learning**:
    - Neural Networks Types
    - Learning Rules
- **Unsupervised Learning**: Exploration of self-organizing maps and other unsupervised techniques.
- **Advanced Architectures**:
    - **CNN (Convolutional Neural Networks)**: Deep learning for visual imagery.
    - **LSTM (Long Short-Term Memory)**: Recurrent neural networks for sequence prediction.
- **Mathematical Foundations**:
    - **Tensors**: Essential data structures for deep learning.

## Repository Structure
This repository is organized into the following main sections:

- **Neural Network Conferences**: Contains LaTeX presentations and conference materials for each topic.
- **Examples**: Practical examples and code snippets (e.g., MATLAB, Markdown) to demonstrate key concepts.
